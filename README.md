Gulp based converter. Converts images and fonts into new formats.

## Setup
Install gulp-cli if you do not have it before.
```
npm i -g gulp-cli
```


## Usage
Convert .png, .jpg photos into .webp.
```
npm run webp
```

Convert .eot, .ttf into .woff, .woff2.
```
npm run woff
```
